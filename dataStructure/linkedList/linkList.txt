

    Link List ----->

        - Linked list is a linear data structure that includes a series of a connected nodes 
           Each node consist of a data value and a pointer that next to the next node
    
        - The list element can be easily inserted or removed without reallocation or 
          reorganization of the entire structure

    Usage of link list
    
        - All application of both stack and queue are application of link list
